# 16.3 マングリング

C++プログラムをコンパイルする段階でマングリングという処理がされます。マングリングとは、関数のオーバーロードという機能がC++にある事に起因しています。

例えば、「16.2 エントリポイント」で取り上げたC言語のコードをエントリポイントを指定してコンパイルしましょう。
```cpp
#include<stdio.h>
int hoge()
{
	puts("hoge only");
}
```
```cpp
gcc -e hoge() a.c
```
コンパイルに成功します。さて、では上記のC言語コードを、C++としてコンパイルを行って見ましょう。
```cpp
g++ -e hoge() a.c
```
